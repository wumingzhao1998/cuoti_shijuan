# 打包成 exe 文件说明

## 方法一：使用 PyInstaller（推荐用于分发）

### 1. 安装 PyInstaller
```bash
pip install pyinstaller
```

### 2. 打包命令
```bash
pyinstaller --onefile --windowed --name="错题生成试卷" --add-data="app.py;." run_app.py
```

### 3. 获取 exe 文件
打包完成后，exe 文件在 `dist` 目录中。

**注意**：Streamlit 应用打包后体积较大（可能 200MB+），且首次启动可能较慢。

---

## 方法二：使用批处理文件（推荐用于个人使用）

直接双击 `启动程序.bat` 即可启动，无需打包。

**优势**：
- 启动速度快
- 文件体积小
- 易于更新和维护

---

## 方法三：使用 Nuitka（体积更小）

### 1. 安装 Nuitka
```bash
pip install nuitka
```

### 2. 打包命令
```bash
python -m nuitka --onefile --windows-disable-console --enable-plugin=streamlit --include-module=streamlit run_app.py
```

---

## 推荐方案

对于个人使用，**推荐使用方法二（批处理文件）**，简单可靠。

如果需要分发给他人使用，且对方没有安装 Python，可以使用方法一或方法三。
